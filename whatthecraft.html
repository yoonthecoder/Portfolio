Front-end Development

What The Craft Website

Project Overview

This project is to create a website for the fictional craft beer festival, 'What The Craft' that our team created. The original purpose of this project was to implement a design and promote the event itself, and our team created the website using WordPress. After some time, I personally wanted to challenge myself and decided to build this website from scratch using HTML, CSS and JavaScript with the help of Bootstrap.

Timeline Jan 2021 - Feb 2021
Role Front-end Web Developer
Tools HTML, CSS, JavaScript, Bootstrap
01

Planning

Based on the wireframes built from the previous project, I spent some time to analyzing how I would divide sections and mark things up. Also, I stored global values such as primary colors used and global spacing as CSS variables to work in a more convenient way.

02

Implement

To better understand users, I defined 3 main user archetypes and set one of them as a major persona. Based on the persona, I created a journey map to understand the whole experience from the user’s perspective by visualizing the process she goes through. It helped me to get a sense of the user’s motivations, needs or some pain points and also discover some opportunities to focus on going forward.

03

Sketch

Before going through the actual design work, I brainstormed all the ideas I gathered from the previous stages and started with some free-sketching of the key pages based on some challenges and the task flows which are covered earlier.
