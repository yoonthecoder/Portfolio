What The Craft Website

Project Overview

This project is to create a website for the fictional craft beer festival, 'What The Craft' that our team created. The original purpose of this project was to promote the event itself by creating digital marketing materials for social media and building a website, and our team decided to create the website using WordPress back then. After some time, I personally wanted to challenge myself as an aspiring developer and decided to build this website from scratch using HTML, CSS and JavaScript with the help of Bootstrap to practice implementing designs in code.

Timeline Jan 2021 - Feb 2021
Role Front-end Web Developer
Tools HTML, CSS, JavaScript, Bootstrap
01

Planning

Based on the wireframes created from the previous project, I spent some time analyzing how I would divide sections for writing HTML Markup. Also, in order to keep the consistency between all the components and not to copy and paste the same styles over and over again, I placed main color palette, default font-size and some spacing in CSS variables ahead of time.

02

Implement

After finishing up HTMl markup and styling using CSS, I added some animations using custom Javascript, Jquery code and integrated AOS library for scrolling fade effect.

03

Test

Lastly, in order to make sure the website is responsive and working properly in any environments, I conducted cross browser testing using Browserstack and checked how the website looks like on different browsers and devices.
