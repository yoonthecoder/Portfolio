Recipe Search App

Project Overview

This project is to demonstrate my working knowledge in getting data using the third-party APIs and create a simple recipe search app by displaying the data properly.

Timeline Feb 2021
Role Web Designer, Web Developer
Tools Adobe XD, HTML, CSS, JavaScript, TheMealDB API
01

Plan

First, I spent some time looking for some recipe APIs that provide a variety of recipes and ingredients. Since I wanted to build the feature that allows users to search meal options for the ingredients they have and check the full recipes of the meal they choose. After doing some research, I decided to go with TheMealDB as it provides some endpoints including both for searching meals by ingredients and getting full recipes of them. Once I decided on the endpoints I would like to use, I checked if they are working as intended using Postman.

02

Design

After that, I worked on creating wireframes and mock-ups for the app to figure out how and where I should display the data in order to ensure a seamless user experience.

03

Application

Based on the plans I made in the previous phases, I started coding to implement the design. First, I structured my HTML and styled it using CSS and then received the JSON data from the endpoints I chose using fetch() method. As the response object returned is not the actual JSON (just an HTTP response), I had to use the json() method to extract the JSON body content from it.
