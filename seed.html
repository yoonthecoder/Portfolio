Front-end Development

SEED WordPress Website

Project Overview

This project is to create a website for the fictional African healthcare charity, ‘SEED’ that our team created. This purpose of this project is to demonstrate my knowledge of building a website using WordPress and installing a third-party plugin.

Timeline March 2020 - May 2020
Role Web Designer, Web Developer
Tools WordPress, Adobe Photoshop
01

Planning

First, I spent some time figuring out the content needed and deciding on a look and feel of the website by creating a style tile that includes logo variations, typography and main UI elements.

02

Wireframing

Based on the plan I made, I created low fidelity wireframes to figure out how I wanted to place and position elements. This process was a major step of this project as I had to choose the most appropriate WordPress theme that would be a good fit for my website.

03

Application

With all the assets and wireframes ready, I started to look for the best theme to implement the website I planned out. I decided to go with the EcoHosting theme as it comes with a carousel banner I wanted to include.

04

Plugin Installation

As this website is for the charity, a donation page is necessary to help potential donors make a donation in an easier and secure way. I decided to install the third-party plugin called ‘GiveWP’ to add online donation forms and customize it.

05

Adding Custom CSS

Lastly, as I wanted to customize sizing and spacing of some sections, such as primary menu on the homepage, I looked for the target element’s id and classes and gave them custom padding, and margin.
