Front-end Development

Note-taking App

Project Overview

This project was to build a simple note-taking app where the user can create and delete notes and search them by title. The app was built using React JS, CSS and Material UI.

Timeline March 2021
Role UI Designer, Web Developer
Tools Figma, React JS, Material UI
01

Plan

First, I spent some time coming up with the key functionalities I wanted to include and how I would implement them by trying out some popular note-taking apps out there. I decided to add the basic features of most note-taking apps that include creating, deleting and searching notes by title.

02

Design

I then created wireframes and mockups using Figma to figure out how I would position the elements and style them to ensure a seamless user experience on the app. This process also helped me to get a sense of how I should separate those UI elements into components.

03

Implement

Based on the plan I made in the previous stage, I separate elements into three components that include App, Sidebar and Main. 'App' is the parent component that includes both 'Sidebar' and 'Main', and also have all the Hooks (useState) to keep track of the states of notes. 'Sidebar' component is the section where users can create, delete and search the notes and also see the list of notes. 'Main' component updates the active note that got clicked in the sidebar and displays the data for the certain note data passed down from the parent component through props.
